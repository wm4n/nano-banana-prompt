服務版本比較方案

同一個 prompt 有 Nano Banana 和 GPT Image 兩個版本時，detail page 如何呈現？

A

並排比較（Side by Side）

Detail page 上方兩欄並排，左邊 Nano Banana 生成圖、右邊 GPT Image 生成圖，清楚對照視覺差異。卡片封面用第一張圖（或最新服務的圖）。

🍌 Nano Banana
🤖 GPT Image
Nano Banana 版
GPT Image 版
B

Tab 切換（Tabbed View）

Detail page 圖片區有 tab，點 "Nano Banana" / "GPT Image" 切換查看對應生成圖。適合圖片較大時保持版面整潔。卡片封面用 cover（預設服務的圖）。

🍌 Nano Banana
🤖 GPT Image
顯示選中服務的圖片
C

垂直堆疊（Stacked）

Detail page 依序展示各服務版本，每個區塊有服務名稱標題 + 對應圖片。類似現有的多圖 prompt 格式（001 號目前就有多張圖），不需要改動太多現有結構。

🍌 Nano Banana
🤖 GPT Image
